This plugin integrates The Feedback Company reviews into a Wordpress website.  Dutch language only.

== Description ==

This plugin requires an API key from The Feedback Company to function.  Using an API key, you can display widgets with customer reviews on your Wordpress website.  You can choose from several widgets.

After installation of the plugin, you will find an overview of the widgets and corresponding shortcodes on the settings screen.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/the-feedback-company` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Obtain an API key from The Feedback Company
4. Use the Settings->The Feedback Company screen to configure the plugin
5. Add the widget to your website via Appearance or the shortcodes.
